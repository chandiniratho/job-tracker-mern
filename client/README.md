# Job Application Tracker (MERN)

A full-stack MERN application to track job applications with authentication, dashboards, filters, charts, and CRUD operations.

## Features
- JWT Authentication (Login/Register)
- Protected Routes
- Add / Edit / Delete Jobs
- Status Filters
- Charts Overview
- Dark Mode
- Pagination

## Tech Stack
- React + TypeScript
- Node.js + Express
- MongoDB Atlas
- JWT Authentication
- CSS Modules

## Setup
### Backend
cd server
npm install
node server.js

### Frontend
cd client
npm install
npm start
